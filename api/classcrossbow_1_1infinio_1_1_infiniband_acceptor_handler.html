virtual

Handle a new incoming connection.

The class is responsible for taking ownership of the InfinibandSocket object and either has to accept or reject the connection request. Otherwise the connection will stay open.

After accepting the connection, it is not yet in a fully connected state so any write operations on the socket will fail until the onConnected callback is invoked on the socket.

The reported length of the private data string may be larger than the actual data sent (as dictated by the underlying transport). Any additional bytes are zeroed out.
