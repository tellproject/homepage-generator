Detailed Description

template<class Context>
class tell::db::TransactionFiber< Context >

Wrapper class around a transaction context.

This class is used to run transactions. It will run in a fiber and can be used to share information between the fiber and the main thread. The fiber can use Transaction<Context>::block to block its execution and the main thread can use Transaction<Context>::wait and Transaction<Context>::unblock to wait for the fiber and unblock its execution
