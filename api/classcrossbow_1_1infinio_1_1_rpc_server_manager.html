Detailed Description

template<typename Manager, typename Socket>
class crossbow::infinio::RpcServerManager< Manager, Socket >

Connection manager base class to manage RPC server connections.

A RPC server connection manager has to implement the createConnection(InfinibandSocket socket, const crossbow::string& data) function to handle create a new implementation specific Connection object associated with the socket.
