Detailed Description

template<typename Handler>
class crossbow::infinio::BatchingMessageSocket< Handler >

Socket wrapper for sending and receiving messages.

The implementation transparently batches multiple messages into one Infiniband buffer to increase throughput at the cost of latency. Messages are flushed every time all events in the poll round have been processed.
